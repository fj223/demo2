## 🚀 使用方法

### 快速使用
```powershell
# 双击运行
run_conversion.bat

# 或命令行
venv\Scripts\python.exe convert_slides_universal.py 文件.html
```

### 转换新文件
```powershell
# Q&A 风格（有隐藏内容）
venv\Scripts\python.exe convert_slides_text.py 新文件.html

# 标准幻灯片
venv\Scripts\python.exe convert_slides_universal.py 新文件.html
```
