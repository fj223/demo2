## 🚀 如何使用

### 方法 1: 一键运行（推荐）

双击运行：
```
run_conversion.bat
```

这会自动：
- 检查环境
- 安装浏览器驱动（如果需要）
- 转换 GPU.html 为可编辑 PPTX

### 方法 2: 手动运行

#### 在 PowerShell 中：

```powershell
# 直接运行（使用虚拟环境中的 Python）
.\venv\Scripts\python.exe convert_slides_text.py 01042026\01042026\GPU.html

# 或者先激活虚拟环境
.\venv\Scripts\Activate.ps1
python convert_slides_text.py 01042026\01042026\GPU.html
```

#### 在 CMD 中：

```cmd
# 直接运行
venv\Scripts\python.exe convert_slides_text.py 01042026\01042026\GPU.html

# 或者先激活虚拟环境
venv\Scripts\activate.bat
python convert_slides_text.py 01042026\01042026\GPU.html
```

### 方法 3: 使用激活脚本

双击运行 `activate_venv.bat`，然后在打开的命令行中：

```cmd
python convert_slides_text.py 01042026\01042026\GPU.html
```

## 🎯 常用命令

### 激活虚拟环境

## 🔧 故障排除

### 问题：PowerShell 无法运行脚本

**错误信息**：
```
无法加载文件 venv\Scripts\Activate.ps1，因为在此系统上禁止运行脚本。
```

**解决方法**：
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

然后重试。

### 问题：找不到 python 命令

**原因**：虚拟环境未激活

**解决方法**：
- 使用完整路径：`.\venv\Scripts\python.exe`
- 或者先激活虚拟环境

### 问题：playwright 浏览器未安装

**解决方法**：
```powershell
.\venv\Scripts\playwright.exe install chromium
```
