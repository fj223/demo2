# HTML 幻灯片转 PPTX 工具

## 🎯 功能

将 HTML 演示文稿转换为**可编辑文字**的 PPTX 文件。

支持的 HTML 类型：
- ✅ GPU.html 风格（Q&A 问答式，带隐藏内容）
- ✅ GIMP.html 风格（标准幻灯片）
- ✅ unittest.html 风格（标准幻灯片）

## 🚀 快速使用

### 方式 1：一键运行（推荐）

双击运行：
```
run_conversion.bat
```

### 方式 2：命令行运行

```powershell
# 转换单个文件
venv\Scripts\python.exe convert_slides_text.py 文件路径.html

# 转换多个文件
venv\Scripts\python.exe convert_slides_universal.py 文件1.html 文件2.html

# 批量转换整个目录
venv\Scripts\python.exe convert_slides_text.py 目录路径
```

## 🔧 转换器说明

### convert_slides_text.py
- **适用于**：GPU.html（Q&A 问答式）
- **特点**：自动点击展开按钮，提取隐藏内容
- **输出**：每个问题一页，包含理论、任务、解决方案

### convert_slides_universal.py
- **适用于**：GIMP.html、unittest.html（标准幻灯片）
- **特点**：提取所有幻灯片内容
- **输出**：每张幻灯片一页

## 💡 使用技巧

### 转换新文件

```powershell
# Q&A 风格（带隐藏内容）
venv\Scripts\python.exe convert_slides_text.py 新文件.html

# 标准幻灯片
venv\Scripts\python.exe convert_slides_universal.py 新文件.html
```

### 批量转换

```powershell
# 转换目录中所有 HTML
venv\Scripts\python.exe convert_slides_universal.py 01042026\01042026\*.html
```

## 🔧 故障排除

### 问题：转换失败

**检查**：
1. 虚拟环境是否激活
2. HTML 文件路径是否正确
3. 查看错误信息

**解决**：
```powershell
# 重新安装依赖
venv\Scripts\pip.exe install -r requirements.txt
```

### 问题：内容不完整

**原因**：HTML 结构特殊

**解决**：
- GPU.html 风格 → 使用 `convert_slides_text.py`
- 其他风格 → 使用 `convert_slides_universal.py`
